What's an API key?

An API key lets this extension talk to an AI model to understand your questions and find the right resources.

If you already have an account with OpenAI, Anthropic, or DeepSeek, use that. If not, DeepSeek is the cheapest option — most queries cost fractions of a cent.

To get a key:

  • DeepSeek — platform.deepseek.com
  • OpenAI — platform.openai.com
  • Claude — console.anthropic.com

Sign up, create a new key, and paste it above. Your key stays on your device — it's never sent anywhere except the AI provider.
